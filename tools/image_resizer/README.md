A estrutura dos diretórios deve ser a seguinte:

|--- redim_images.py
|--- images
     |--- beignets
	  |--- image00001.jpg
	  |--- image00002.jpg
	  |--- image00003.jpg
	  |--- image00004.jpg
          |--- ...
     |--- bruschetta
     |--- hamburger
     |--- ...

Então basta rodar o programa como $>python redim_images.py -d images
